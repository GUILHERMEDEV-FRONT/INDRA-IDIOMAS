## 📋 Sobre o Projeto

O **Indra Idiomas** é um projeto frontend desenvolvido com **React** e **TypeScript**, utilizando o **Vite** como ferramenta de build para garantir a máxima performance e rapidez durante o desenvolvimento.

A aplicação foi estruturada seguindo o conceito de componentes modulares reutilizáveis, onde cada seção da página principal (Landing Page) foi isolada para facilitar a manutenção e garantir uma arquitetura de código limpa. O projeto conta com seções explicativas sobre cursos, livros digitais (Ebooks), galeria de estudantes, serviços de tradução e formulários de contato para captação de clientes.

---

## 🚀 Funcionalidades da Interface

O projeto foi totalmente modularizado em componentes independentes:

* **Seção Hero Premium (`Hero.tsx`):** Apresentação de impacto da escola de idiomas com chamadas para ação (CTA) estratégicas.
* **Apresentação Institucional (`AboutUs.tsx`):** História, valores e o propósito da marca Indra Idiomas.
* **Vitrine de Cursos (`Courses.tsx`):** Listagem dinâmica e elegante dos idiomas e modalidades oferecidas.
* **Módulo de Ebooks (`Ebooks.tsx`):** Divulgação de materiais didáticos e livros digitais exclusivos.
* **Galeria de Alunos (`StudentGallery.tsx`):** Espaço visual dedicado à comunidade e experiências reais dos estudantes.
* **Área de Tradução (`Translation.tsx`):** Informações detalhadas sobre serviços de tradução e proficiência.
* **Formulário de Contato (`ContactForm.tsx`):** Interface intuitiva e validada para captação de leads e novos alunos.
* **Central de Dúvidas (`Faq.tsx`):** Seção de perguntas frequentes no estilo *accordion* para melhorar a experiência do usuário (UX).

---

## 🛠️ Tecnologias Utilizadas

* **React.js (v18+):** Biblioteca principal para a construção da interface baseada em componentes reutilizáveis.
* **TypeScript:** Adição de tipagem estática rigorosa para evitar erros em tempo de execução e melhorar a produtividade no editor.
* **Vite:** Bundler ultra-rápido focado em performance, substituindo o antigo Create React App.
* **CSS3 Dinâmico:** Estilização customizada aplicada de forma global e local para garantir um layout responsivo e moderno.
